谷歌邮箱别名设置

2026年06月23日
网络智能
留言

Gmail支持四种方式实现一账号多收件地址：一是利用点号忽略特性，如your.name@gmail.com与yourname@gmail.com等效；二是加号标签法，如yourname+news@gmail.com；

三是添加并验证备用邮箱作为发送别名；四是通过Google Workspace为自定义域名添加无限别名。
谷歌邮箱怎么设置别名 实现一个账号多个收件地址使用教程

如果您拥有一个谷歌邮箱（Gmail）主账号，但希望使用多个不同的电子邮件地址接收邮件，Gmail 提供了“别名”功能，允许您在不创建新账号的前提下扩展收件地址。以下是实现一个账号多个收件地址的具体操作步骤：

一、利用 Gmail 的“点号忽略”特性添加别名
Gmail 会自动忽略邮箱地址中用户名部分的英文句点（.），这意味着 you.name@gmail.com、youname@gmail.com 和 y.o.u.n.a.m.e@gmail.com 实际都指向同一个收件箱。该特性无需任何设置，系统默认启用，所有发往这些变体地址的邮件均会自动归入主账号。

1、确认您的主 Gmail 地址为 yourname@gmail.com（以实际为准）。

2、对外提供任意带点号分隔的变体地址，例如 your.name@gmail.com 或 y.o.u.r.n.a.m.e@gmail.com。

3、收到邮件后，可在收件箱中直接查看，无需额外配置或转发规则。

二、通过“加号标签”（+alias）方式创建动态别名
Gmail 支持在用户名后添加加号（+）及任意字符作为标签，所有发往 yourname+label@gmail.com 的邮件均会送达 yourname@gmail.com 收件箱。此方法可用于分类识别来源，且无需预设或验证。

1、确定主邮箱为 yourname@gmail.com。

2、向他人提供带 + 号的地址，例如 yourname+newsletter@gmail.com 或 yourname+shopping@gmail.com。

3、登录 Gmail 后，在搜索栏输入 to:yourname+newsletter@gmail.com 即可筛选对应来源的全部来信。

三、添加已验证的备用电子邮件地址作为发送别名
此方法允许您从同一 Gmail 账户界面，以不同邮箱地址（如公司邮箱或自定义域名邮箱）对外发送邮件，同时仍能统一管理收件。需提前验证该备用地址所有权。

1、登录 Gmail 网页版，点击右上角头像 → “管理您的 Google 账户”。

2、左侧菜单选择“个人信息” → “联系信息” → “电子邮件” → “添加其他电子邮件地址”。

3、输入待添加的邮箱（例如 contact@yourdomain.com），点击“下一步”后按提示完成验证码验证。

4、返回 Gmail 设置 → “查看所有设置” → “账户和导入” → “发送邮件”，勾选刚验证的地址，并设置默认回复地址。

四、使用 Google Workspace 自定义域名邮箱（需付费订阅）
若您已注册并验证了自有域名（如 yourcompany.com），可通过 Google Workspace 订阅服务，为同一账户开通无限数量的 @yourcompany.com 邮箱别名，所有邮件统一汇聚至主 Gmail 收件箱，且支持独立签名与发信身份。

1、访问 workspace.google.com，注册或登录已有 Workspace 管理控制台。

2、进入“用户”页面，选择主账号 → “管理用户邮箱别名” → “添加别名”。

3、输入新别名（例如 support@yourcompany.com 或 hello@yourcompany.com），保存后即时生效。

4、无需额外配置收信规则，所有发往该别名的邮件自动进入主账号收件箱。
